# Stokvel Management System

## 🚀 Overview
A full-stack web application that helps users manage stokvel (group savings) activities, track contributions, and monitor financial progress.

## 🛠 Tech Stack
- Frontend: Angular
- Backend: Node.js / Flask
- Database: SQL
- Cloud: Azure / AWS (optional if used)

## ✨ Features
- User authentication and login
- Create and manage stokvel groups
- Track member contributions
- Admin dashboard with analytics
- Proposal/financial tracking system
